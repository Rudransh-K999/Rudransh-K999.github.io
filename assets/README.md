# Global Assets Store

Upload your custom audio files here. The application is configured to look for these specific filenames:

- `bgm.mp3`: Background music.
- `click.wav`: Navigation/Button hover sound (also supports `.mp3` if renamed).
- `hold.mp3`: The sound played during the "Establish Link" hold process.
- `pfp.jpg`: Your profile picture.

Note: If a local file is missing or fails to load, the system will automatically fall back to the default remote assets.
